# 🍕 Pizza Sales Analytics & Visualization

Proyek ini adalah sebuah studi kasus Analisis Data (Data Analytics) dan Visualisasi Data secara *end-to-end* menggunakan dataset penjualan restoran pizza. Tujuan dari proyek ini adalah untuk menggali wawasan (*insights*) bisnis dari data mentah dan menyajikannya dalam bentuk dasbor (*dashboard*) interaktif yang mudah dipahami.

## 🛠️ Alat & Teknologi (Tech Stack)

Proyek analisis ini dibangun menggunakan alat bantu berikut:

* ![Power BI](https://img.shields.io/badge/Power_BI-F2C811?style=for-the-badge&logo=powerbi&logoColor=black) **Microsoft Power BI:** Digunakan untuk pemrosesan data (Power Query), pembuatan *measure* (DAX), dan perancangan dasbor visualisasi utama.
* ![Excel](https://img.shields.io/badge/Excel-217346?style=for-the-badge&logo=microsoftexcel&logoColor=white) **Microsoft Excel:** Digunakan untuk eksplorasi data awal dan pembuatan *Data Model*.

## 📂 Struktur Repositori

Berikut adalah penjelasan mengenai *file* yang ada di dalam repositori ini:

* `Source Data Pizza Sales.txt` - Merupakan *file* sumber data mentah (raw data) yang digunakan untuk analisis.
* `Data Model - Pizza Sales.xlsx` - *File* Excel yang berisi pemodelan data (*Data Modeling*) dan relasi antar tabel sebelum divisualisasikan.
* `Insight Pizza Sales.pbix` - *File* utama *project* Power BI yang memuat langkah transformasi data, perhitungan DAX, serta dasbor interaktif.
* `Page 1.png` & `Page 2.png` - Tangkapan layar (*screenshot*) statis dari halaman dasbor yang telah selesai dibuat.
